# Leon web browser
> just simple web browser designed by python and QT

![](view.png)


## Setup
1. Clone this repo in your local dir.

2. Head to repo dir, then in your terminal:
```sh
python main.py
```


## Modules you should have in pip list

- PyQt5.QtWidgets
- PyQt5.QtCore
- PyQt5.QtWebEngineWidgets
- OR use Anaconda env
